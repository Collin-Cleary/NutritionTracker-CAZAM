## Minimum Requirements
node 18.12.1

mongodb 7.0.7

npm 8.19.2


## How to run it

1. clone our repository
2. open in your preferred coding text editor
3. cd into the backend folder
4. run npm install
5. run "node server.js" (or "npm test")
